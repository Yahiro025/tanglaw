# Project Instructions

## Design Context

**Register**: Product
**Creative North Star**: "The Guiding Beacon"
**Brand Personality**: Guiding, Reliable, Professional.
**Strategic Principles**:
- **Guiding Light**: Maintain absolute clarity on user progress and next steps.
- **Empathetic Reliability**: High-precision data wrapped in a supportive tone.
- **Dynamic Utility**: Functional but "alive" with thoughtful transitions and feedback.

For visual details, tokens, and components, see [DESIGN.md](./DESIGN.md).
For product strategy and anti-references, see [PRODUCT.md](./PRODUCT.md).
