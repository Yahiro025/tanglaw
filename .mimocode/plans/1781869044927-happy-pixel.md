## Content Structure

### 1. Title & Brief Introduction
- Project name: TANGLAW (Tagalog for "light" / "illumination")
- One-liner: AI-powered scholarship navigation portal for Filipino tertiary students
- Core purpose: Helps students find, understand, and apply for scholarships

### 2. Problem Section
- Filipino tertiary students face fragmented, hard-to-navigate scholarship information
- 30.5% of Grade 3 learners show basic reading proficiency
- 0.47% of Grade 12 learners demonstrate grade-level readiness
- Students lack access to verified grant sources, eligibility summaries, and coaching tools in one place
- Scholarship research is noisy, fragmented, and difficult to navigate

### 3. Solution Section
- TANGLAW is a full-stack application with Next.js frontend, Express backend, PostgreSQL database, LangChain AI integration, and Supabase hosting
- Combines scholarship directory, readiness assessment, exam reviewer, and AI chatbot companion ("Owel")
- Users can register, browse scholarships, take quizzes, chat with AI, and track progress from a dashboard
- Focuses on centralized scholarship discovery with AI-guided navigation

### 4. Technologies Used
- **Frontend:** Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS v4, Framer Motion, lucide-react
- **Backend:** Express.js, TypeScript (CommonJS), Prisma v7 ORM
- **Database:** PostgreSQL (Supabase), pgvector for embeddings
- **AI/LLM:** Google Gemini 3.1 Flash-Lite (primary), OpenRouter free models (fallback), LangChain
- **Auth:** NextAuth.js v4 (CredentialsProvider), JWT (jsonwebtoken + bcryptjs)
- **Testing:** Vitest (unit), Playwright (E2E)
- **Deployment:** Vercel (frontend), Render (backend), Supabase (database)

### 5. Key Features
- Scholarship browser with search, filtering (income, sector, program), and eligibility recommendations
- Readiness assessment quiz (Math, Science, English, Filipino, Logic)
- Exam reviewer module
- AI chatbot companion ("Owel") with RAG-powered scholarship search
- Secure student dashboard with NextAuth.js authentication
- Responsive, animated UI with Framer Motion

### 6. How to Run the Project
- **Prerequisites:** Node.js 20+, npm, Supabase account
- **Backend setup:** install, prisma generate, prisma db push, seed, dev server
- **Frontend setup:** install, dev server
- **Optional checks:** typecheck, lint, unit tests, E2E tests
- No demo link (per user preference)
