# Plan: Restyle navbars to match Landas pill-shaped glassmorphism style

## Goal
Restyle the public SiteHeader and dashboard header to match Landas's centered, floating pill-shaped glassmorphism navigation bar.

## Files to modify

### 1. `src/components/site-header.tsx` (public pages)
**Current:** Full-width header with border-bottom, logo left, nav center-right, ThemeChanger right.

**Target (Landas style):**
- **Floating pill shape**: Center the nav in a `rounded-full` pill with glassmorphism (`backdrop-blur-xl`, semi-transparent bg, subtle border)
- **Position**: `fixed top-0 left-0 right-0 z-50 flex justify-center pt-4 px-4`
- **Logo**: Show "TANGLAW" wordmark on non-home pages (left of pill, like Landas), hide on home
- **Nav links**: Inside the pill, `rounded-full` items with active = `bg-primary/15 text-primary`, hover = subtle bg
- **Auth buttons**: Divided by a subtle `v-divider`, Log In = ghost, Sign Up = highlighted pill
- **Scroll behavior**: Keep auto-hide on scroll down / show on scroll up
- **Mobile**: Keep hamburger but style it as a pill button

### 2. `src/app/dashboard/layout.tsx` (dashboard pages)
**Current:** Full-width header with border-bottom, logo + nav left, ThemeChanger + Sign Out right.

**Target (Landas style):**
- **Same floating pill shape** as public header
- **Nav links**: Overview, Scholarships, Readiness inside the pill
- **Sign Out**: Styled as a subtle button inside or after the pill
- **Keep ThemeChanger** accessible

## Key styling tokens (adapted from Landas to TANGLAW's blue/purple theme)

| Landas | TANGLAW equivalent |
|--------|-------------------|
| `bg-leaf-dark/60` | `bg-[color:var(--theme-surface)]/60` |
| `border-gold/10` | `border-white/10` |
| `backdrop-blur-xl` | `backdrop-blur-xl` |
| `text-gold/50` (inactive) | `text-[color:var(--theme-typography-secondary)]` |
| `bg-gold/10 text-gold/90` (active) | `bg-primary/15 text-primary` |
| `hover:bg-gold/8` | `hover:bg-white/5` |
| `font-magical` (logo) | `font-display` (already used) |
| `text-landas-gradient` (logo) | `bg-gradient-to-r from-primary to-[color:var(--theme-accent-periwinkle)] bg-clip-text text-transparent` |

## Implementation details

### SiteHeader changes:
1. Remove the `border-b` and full-width bg from `<header>`
2. Wrap nav links in a centered pill container:
   ```
   <nav className="flex items-center gap-1 rounded-full border border-white/10 bg-[color:var(--theme-surface)]/60 backdrop-blur-xl px-3 py-2.5 shadow-lg shadow-black/10">
   ```
3. Style each NavLink as `rounded-full px-3.5 py-1.5 text-[10px] uppercase tracking-[0.18em]` with active/hover states
4. Add subtle bottom glow line on active links (like Landas)
5. Add a thin vertical divider between nav groups
6. Move "TANGLAW" wordmark to float left of the pill (hidden on home, visible on other pages)
7. Keep ThemeChanger inside or adjacent to the pill

### Dashboard layout changes:
1. Same pill container approach
2. Nav links: Overview, Scholarships, Readiness
3. Sign Out button styled as a subtle pill after a divider
4. Keep the existing mobile hamburger pattern but restyle to match pill aesthetic
